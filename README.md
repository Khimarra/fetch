#### Endpoints

- GET `/user`
	- Index route returning an array of all Users
- POST `/user`
	- Create route for a new user
- GET `/user/:id`
  - Show route for a user requested by ID
- PUT `/user/:id/`
  - Edit a user's profile info
- DELETE `/user/:id`
  - Delete a user's profile
